# 1003번

## 주어진 fibonacci 함수에서 0과 1이 출력되는 횟수 구하기
- 이전에 리턴된 값을 저장하려면 리스트 이용
- 리스트의 인덱스 번호 n == fibonacci(n) --> ex: [2]번 항목은 n=2일 때 0, 1이 리턴된 수
- 0과 1이 리턴된 수를 각각의 리스트로 나누어 구하기
